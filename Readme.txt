




To follow along with the examples in this book, the reader will need to have an Apple computer with OS X 10.10 or higher installed. 
They will also need to install Xcode version 7.0 or higher with Swift version 2 or higher and have at least basic knowledge of the Swift programming language.
The examples, as they are shown in this book, should work with the Linux port of the Swift language however the downloadable code for this book requires Xcode 
which is only available on the Apple platform.
